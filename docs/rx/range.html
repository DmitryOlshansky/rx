    Module rx.range

    This module defines some operations like range.

    Example

    Overview 

    import rx.subject;
import std.algorithm : equal;
import std.array : appender;
import std.conv : to;

auto subject = new SubjectObject!int;
auto pub = subject.drop(2).take(3);

auto buf = appender!(int[]);
auto disposable = pub.subscribe(observerObject!int(buf));

foreach (i; 0 .. 10)
{
    subject.put(i);
}

auto result = buf.data;
assert(equal(result, [2, 3, 4]));

    Functions

    Name Description
    drop Creates the observable that results from discarding the first n elements from the given source.
    take Creates a sub-observable consisting of only up to the first n elements of the given source.
    takeLast Creates a observable that take only a last element of the given source.
